### 逻辑判断

##### 判断
~~~bash
# 写法一
test <表达式> ;
# 写法二
[ <表达式> ] ;
# 写法三
[[ <表达式> ]] ;
~~~

> 以上三种均是等价的,如果表达式为真,test命令成功执行,返回值为0; 如果表达式为假,test命令执行失败,返回值为1  
> `[` 和`]`与内部表达式之间必须有空格  
> 第三种写法支持正则表达式  
> 在使用`(`和`)`时候必须转义

##### 算术判断
~~~bash
(( <表达式> )) ;
~~~
> 表达式非零为真,零为假
